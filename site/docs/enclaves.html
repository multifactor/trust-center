Verify a AWS Nitro attestation document. Validates the attestation document signature and certificate. NOTE: You still need to verify the PCRs yourself to check if the request is generated by the desired enclave image.
Example
const trust = require('trust-center');
trust.enclaves.nitro.verify(validDocument); // -> {valid: true, attr: {...}}
trust.enclaves.nitro.verify(invalidDocument); // -> {valid: false, reason: '...'}

document Buffer AWS Nitro attestation document as a Buffer
Returns:
The validation result and attestation document attributes or rejection reason
